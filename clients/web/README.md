# Web 客户端

基于 React 19 + TypeScript 的 Web 前端应用。

## 技术栈
- React 19
- TypeScript
- 原生 CSS

## 开发
```bash
# 从项目根目录运行
bun dev
```

## 构建
```bash
bun run build
```

## 文件结构
- `index.tsx` - 应用入口
- `App.tsx` - 主应用组件
- `TodoList.tsx` - 待办事项组件
- `FileTransfer.tsx` - 文件传输组件
- `APITester.tsx` - API 测试组件
- `index.css` - 全局样式
- `index.html` - HTML 模板
